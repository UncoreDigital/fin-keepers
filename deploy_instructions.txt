FINKEEPERS SEO DEPLOYMENT INSTRUCTIONS
=====================================

PRIORITY LEVELS:
P0 = Critical (Deploy immediately)
P1 = High (Deploy within 1 week)  
P2 = Medium (Deploy within 2 weeks)

=== P0 CRITICAL TASKS (DEPLOY IMMEDIATELY) ===

1. UPLOAD CORE SEO FILES:
   - Upload robots.txt to website root: /robots.txt
   - Upload sitemap.xml to website root: /sitemap.xml
   - Verify both files are accessible:
     * https://finkeepers.us/robots.txt
     * https://finkeepers.us/sitemap.xml

2. UPDATE ALL PAGE HEAD SECTIONS:
   Copy and paste the HTML from each file in /updated-head-snippets/ into the <head> section of corresponding pages:
   
   HOMEPAGE (/index.html or equivalent):
   - Copy content from: updated-head-snippets/homepage-head.html
   - Paste into <head> section, replacing existing title/meta tags
   
   BOOKKEEPING PAGE (/book-keeping/index.html):
   - Copy content from: updated-head-snippets/bookkeeping-head.html
   - Paste into <head> section
   
   REPEAT FOR ALL SERVICE PAGES using corresponding head snippet files

3. SUBMIT TO GOOGLE SEARCH CONSOLE:
   - Go to: https://search.google.com/search-console
   - Add property: finkeepers.us (if not already added)
   - Submit sitemap: Sitemaps > Add new sitemap > enter "sitemap.xml"
   - Request indexing for homepage and main service pages

=== P1 HIGH PRIORITY TASKS (WITHIN 1 WEEK) ===

4. IMAGE OPTIMIZATION:
   - Replace all images with WebP versions from /images/ folder
   - Update all <img> tags using snippets from alt_tags.csv
   - Add loading="lazy" to images below the fold
   - Add width/height attributes to prevent layout shift

5. SPEED OPTIMIZATIONS:
   - Upload minified CSS files from /css-js/ folder
   - Add preconnect tags to <head>:
     <link rel="preconnect" href="https://fonts.googleapis.com">
     <link rel="preconnect" href="https://codia-f2c.s3.us-west-1.amazonaws.com">
   
6. MOBILE OPTIMIZATION:
   - Update viewport meta tag:
     <meta name="viewport" content="width=device-width, initial-scale=1.0, viewport-fit=cover">
   - Ensure all buttons are minimum 44px height
   - Test mobile-friendliness: https://search.google.com/test/mobile-friendly

=== P2 MEDIUM PRIORITY TASKS (WITHIN 2 WEEKS) ===

7. CONTENT UPDATES:
   - Add keyword-optimized content blocks from keyword_strategy.md
   - Update H1 and H2 tags as specified in meta_and_head_updates.csv
   - Implement internal linking suggestions

8. ACCESSIBILITY IMPROVEMENTS:
   - Add ARIA labels from accessibility_report.md
   - Implement skip navigation links
   - Add focus indicators for keyboard navigation

9. PERFORMANCE MONITORING:
   - Set up Google Analytics 4 (if not already done)
   - Configure Google Search Console alerts
   - Schedule weekly PageSpeed Insights checks

=== STEP-BY-STEP UPLOAD INSTRUCTIONS ===

FOR CPANEL USERS:
1. Log into cPanel
2. Open File Manager
3. Navigate to public_html folder
4. Upload robots.txt and sitemap.xml to root directory
5. Edit HTML files to add head snippets
6. Upload optimized images to /images/ folder
7. Clear any caching (if using caching plugins)

FOR WORDPRESS USERS:
1. Install Yoast SEO or RankMath plugin
2. Use plugin settings to add meta titles/descriptions
3. Upload robots.txt via plugin or FTP
4. Submit sitemap through plugin
5. Use plugin's schema markup features
6. Install image optimization plugin (Smush, ShortPixel)

FOR NETLIFY/VERCEL USERS:
1. Update your Git repository with new files
2. Push changes to trigger automatic deployment
3. Verify deployment in hosting dashboard
4. Check build logs for any errors
5. Test all URLs after deployment

FOR CLOUDFLARE USERS:
1. Upload files via your hosting provider
2. Purge Cloudflare cache after upload
3. Enable Cloudflare optimizations:
   - Auto Minify: CSS, HTML, JS
   - Brotli compression
   - Polish (image optimization)
4. Set browser cache TTL to 1 year for static assets
