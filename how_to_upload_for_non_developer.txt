HOW TO UPLOAD SEO FILES - NON-DEVELOPER GUIDE
==============================================

This guide will help you upload the SEO optimization files to your website without any technical knowledge.

WHAT YOU NEED:
- Access to your website hosting account (cPanel, WordPress admin, or hosting dashboard)
- The SEO files package you downloaded
- About 30-60 minutes of time

=== OPTION 1: CPANEL (MOST COMMON) ===

STEP 1: LOG INTO CPANEL
1. Go to your hosting provider's website
2. Log into your account
3. Look for "cPanel" button and click it
4. You should see a dashboard with many icons

STEP 2: OPEN FILE MANAGER
1. Look for an icon called "File Manager" (usually has a folder icon)
2. Click on "File Manager"
3. A new window will open showing your website files
4. Look for a folder called "public_html" and double-click it
5. You should now see your website files (index.html, images folder, etc.)

STEP 3: UPLOAD ROBOTS.TXT
1. In the File Manager, make sure you're in the public_html folder
2. Look for a button that says "Upload" (usually at the top)
3. Click "Upload" 
4. Click "Select File" or "Choose Files"
5. Find the "robots.txt" file from your SEO package
6. Select it and click "Open"
7. The file will upload automatically
8. Go back to the File Manager (close upload window)
9. You should now see "robots.txt" in your file list

STEP 4: UPLOAD SITEMAP.XML
1. Repeat the same process as Step 3
2. This time select "sitemap.xml" from your SEO package
3. Upload it to the same public_html folder
4. You should now see both "robots.txt" and "sitemap.xml" in your file list

STEP 5: TEST THE FILES
1. Open a new browser tab
2. Go to: https://finkeepers.us/robots.txt
3. You should see text starting with "User-agent: *"
4. Go to: https://finkeepers.us/sitemap.xml
5. You should see XML code with your website URLs
6. If both work, you're done with the basic setup!

=== OPTION 2: WORDPRESS ADMIN ===

STEP 1: LOG INTO WORDPRESS
1. Go to: https://finkeepers.us/wp-admin
2. Enter your username and password
3. You should see the WordPress dashboard

STEP 2: INSTALL SEO PLUGIN (IF NOT ALREADY INSTALLED)
1. Go to "Plugins" > "Add New"
2. Search for "Yoast SEO" or "RankMath"
3. Click "Install Now" then "Activate"

STEP 3: UPLOAD ROBOTS.TXT
1. Go to "SEO" > "Tools" (in Yoast) or "General Settings" (in RankMath)
2. Look for "File Editor" or "Edit Files"
3. Find "robots.txt" option
4. Copy the content from the robots.txt file in your SEO package
5. Paste it into the editor
6. Click "Save Changes"

STEP 4: SUBMIT SITEMAP
1. In your SEO plugin, look for "Sitemaps" or "Sitemap Settings"
2. Make sure XML sitemaps are enabled
3. Copy the sitemap URL (usually: https://finkeepers.us/sitemap_index.xml)
4. We'll use this in the next section

=== OPTION 3: WEBSITE BUILDER (WIX, SQUARESPACE, ETC.) ===

FOR WIX:
1. Log into your Wix account
2. Go to "Marketing & SEO" > "SEO Tools"
3. Look for "robots.txt" and "Sitemap" options
4. Upload or paste the content from your SEO files

FOR SQUARESPACE:
1. Log into Squarespace
2. Go to "Settings" > "Advanced" > "SEO"
3. Look for "robots.txt" and sitemap options
4. Upload or configure as needed

FOR SHOPIFY:
1. Log into Shopify admin
2. Go to "Online Store" > "Preferences"
3. Scroll down to "Search engine listing preview"
4. Edit meta titles and descriptions using the content from your SEO package

FINANCIAL REPORTING PAGE:

(Continue for all pages listed in the CSV file)

=== SUBMITTING TO GOOGLE SEARCH CONSOLE ===

STEP 1: ACCESS GOOGLE SEARCH CONSOLE
1. Go to: https://search.google.com/search-console
2. Sign in with your Google account
3. If finkeepers.us is not already added, click "Add Property"
4. Enter: finkeepers.us
5. Follow verification steps (usually involves adding a meta tag or uploading a file)

STEP 2: SUBMIT SITEMAP
1. Once verified, click on your property (finkeepers.us)
2. In the left menu, click "Sitemaps"
3. Click "Add a new sitemap"
4. Enter: sitemap.xml
5. Click "Submit"
6. Wait a few minutes, then refresh - it should show as "Success"

STEP 3: REQUEST INDEXING
1. In Search Console, click "URL Inspection" (left menu)
2. Enter your homepage URL: https://finkeepers.us
3. Click "Request Indexing"
4. Repeat for your main service pages:
   - https://finkeepers.us/book-keeping
   - https://finkeepers.us/financial-reporting
   - https://finkeepers.us/payroll-assistance
   - https://finkeepers.us/cashflow-management
   - https://finkeepers.us/temporary-cfo

=== TROUBLESHOOTING COMMON ISSUES ===

PROBLEM: "Can't find File Manager in cPanel"
SOLUTION: Look for "Files" section, or contact your hosting provider

PROBLEM: "robots.txt shows 404 error"
SOLUTION: Make sure you uploaded it to the root directory (public_html), not a subfolder

PROBLEM: "Sitemap shows errors in Google Search Console"
SOLUTION: Check that all URLs in the sitemap are accessible and return 200 status

PROBLEM: "Website looks broken after changes"
SOLUTION: Restore from backup (contact hosting provider) or undo recent changes

PROBLEM: "Can't access WordPress admin"
SOLUTION: Check your login credentials or contact your web developer
